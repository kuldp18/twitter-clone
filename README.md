## Features ðŸ”¥

- Fully Responsive
- OAuth Login using Google
- Post Tweet
- Delete Tweet
- Like Tweet
- Post Reply to Tweet
- Tweets are shown with corresponding timestamps
- Images can be also added while tweeting
- Custom emoji picker for tweets
